# ☁️ Weather Forecast App

A React app that displays weather data by fetching data from OpenWeatherMap API.

## 🔍 Features
- Search weather by city
- Displays temperature, condition, etc.
- Handles API errors and loading states

## 🛠️ Tech Stack
- React.js
- CSS3
- Fetch API
- OpenWeatherMap API
